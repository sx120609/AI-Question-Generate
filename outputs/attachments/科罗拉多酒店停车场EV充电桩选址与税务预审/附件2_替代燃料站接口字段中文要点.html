AFDC 替代燃料站 API 字段说明中文要点

来源：https://developer.nlr.gov/docs/transportation/alt-fuel-stations-v1/all/

用途：解释 All Stations API 字段，用于读取站点状态、接入类型、经纬度和公共可达性。

中文要点

  1. All Stations API 可返回站点名称、地址、城市、州、邮编、经纬度、状态、开放日期、更新时间等字段。
  2. EV 相关字段包括 Level 2 数量、DC 快充数量、连接器类型、网络、价格说明和是否工作场所充电。
  3. 公共可达性通常需要结合 Groups With Access Code、Access Code、Restricted Access 等字段判断。
  4. API 字段有缺失和延迟，不能替代现场确认。

使用边界

本文件只说明公开字段口径，不能证明某一站点当前可用或酒店停车场施工可行。
